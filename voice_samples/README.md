# Voice Samples

Put local CosyVoice prompt audio files here.

Example local files:

```text
voice_samples/my_voice.wav
voice_samples/my_voice_prompt.txt
```

Audio files and transcript text files in this folder are ignored by Git, so your private voice sample is not uploaded.

For zero-shot voice cloning, CosyVoice needs:

- `prompt_wav`: a short sample of your voice
- `prompt_text`: the exact words spoken in that sample
